react router es una herramienta que se usa en proyectos con react para manejar la navegación entre diferentes secciones sin que la página se recargue completamente en lugar de usar enlaces tradicionales react router permite cambiar de vista de forma dinámica y rápida utilizando componentes como browserrouter, routes y route también se puede usar un hook llamado usenavigate, que permite redirigir al usuario a otra ruta desde una función por ejemplo con un botón al hacer clic se puede llevar al usuario a otra sección como /albumes esto ayuda a que la aplicación se sienta más fluida
